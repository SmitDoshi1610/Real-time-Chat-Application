# Chat App

## Run Locally

Clone the project

```bash
  git clone git@github.com:Ekam-Preet-Singh/Minor-Project.git
```

Go to the project directory

```bash
  cd Minor-Project
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```

Start the Client

```bash
  cd frontend
  npm start
```
